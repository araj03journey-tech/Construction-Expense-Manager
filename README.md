# Construction Expense Manager 🏗️📊
**اسکریپت پایتون برای ساخت خودکار قالب پیشرفته مدیریت هزینه‌ها و تنخواه‌گردان پروژه‌های عمرانی**

در پروژه‌های عمرانی و ساختمانی، مدیریت دقیق تنخواه‌ها، تفکیک خریدهای کارگاهی از دستمزدها و پیگیری بدهی‌ها یکی از چالش‌های اصلی مهندسین و مدیران پروژه است. این اسکریپت پایتون، یک قالب اکسل (Excel) کاملاً استاندارد، راست‌چین شده و فرمول‌نویسی‌شده تولید می‌کند که تمام نیازهای حسابداری کارگاهی را پوشش می‌دهد.

## ✨ ویژگی‌های این قالب اکسل
این کد فایلی با دو شیت (Sheet) اصلی تولید می‌کند:

۱. **شیت کل هزینه‌ها (دیتابیس خام):**
- دارای ستون‌های مجزا برای ثبت "مبلغ کل فاکتور"، "مبلغ پرداختی" و محاسبه خودکار "مانده بدهی".
- دسته‌بندی هوشمند هزینه‌ها به دو گروه «خرید مصالح» و «دستمزد خدمات».
- ثبت دقیق محل پرداخت (حساب اصلی یا کارت تنخواه‌داران مختلف).
- پیگیری وضعیت فاکتورها و رسیدهای فیزیکی.

۲. **شیت خلاصه وضعیت (داشبورد هوشمند):**
- محاسبه لحظه‌ای و خودکار مجموع پرداختی‌های خرید مصالح.
- محاسبه لحظه‌ای مجموع پرداختی‌های دستمزد کارگران و پیمانکاران.
- نمایش مجموع بدهی‌های کارگاه با رنگ هشدار (قرمز).
- تفکیک دقیق پرداختی‌ها بر اساس هر شخص (تنخواه‌دار شماره یک، تنخواه‌دار شماره دو، حساب اصلی).
